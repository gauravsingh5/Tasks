## Task 1

Create a function/method that takes in a parameter of type String (or an equivalent) and return the sum of all letters ordinal value.

Ordinal Value of a == 1, b == 2 

Pay attention to the case! (a == A)

### Examples

```py
textSum ("Hello") -> 52
textSum ("Bye")   -> 32
textSum ("Sum")   -> 53
```

## Task 2

Create a function/method that takes a parameter of type int and returns the sum of all the primes up to that number.

The parameter is exclusive!

The parameter will be a value specified as 0<n<200.

### Examples

```py
primeSum(16) -> 42
primeSum(19) -> 59
primeSum(5) ->  6
```

## Task 3

Create a function/method that has a parameter of type int and returns the most similar prime in a list of all primes up to the given value.

The parameter is exclusive.

### Examples

```py
similarPrime(71)  -> 17
similarPrime(32)  -> 23
similarPrime(371) -> 137
```
